# GPT-4

## Overview
GPT-4 is OpenAI's large language model, released in March 2023. It is the successor to GPT-3.5 (ChatGPT) and features significantly improved reasoning capabilities.

## Key Capabilities
- Advanced natural language understanding and generation
- Stronger performance on logical reasoning benchmarks vs prior models
- Multimodal capabilities (vision + text)
- Better instruction following and alignment

## Known Weaknesses
- Struggles with out-of-distribution (OOD) reasoning tasks
- Performance degrades on novel problem distributions
- Can produce confident but incorrect logical inferences
